Project Overview

The Mobile Sales Dashboard is a data analytics project developed using Microsoft Power BI and Microsoft Excel to analyze and visualize mobile sales data.

This dashboard helps businesses monitor sales performance, identify top-selling mobile brands, analyze customer purchasing behavior, and track revenue trends through interactive visualizations.

Objectives
Analyze mobile sales performance
Track monthly and yearly sales trends
Identify top-selling brands and products
Study customer purchasing behavior
Generate business insights using data visualization
Create an interactive and user-friendly dashboard

Tools & Technologies

Microsoft Excel
